# MyPets

An ASP.NET MVC based PetCare system that allows users to explore pet services, place orders, and share feedback, with an integrated admin panel.

PetCare ASP.NET MVC Project
This is a PetCare web application developed using ASP.NET MVC.
It includes pet services, order management, and customer feedback system.

Features
Pets Gallery
Pet shop
Pet Services Management
Order System
Customer Feedback
Admin Panel
Technologies Used
ASP.NET MVC
C#
SQL Server
HTML, CSS, Bootstrap
How to Run
Clone the repository
Open project in Visual Studio
Restore NuGet packages
Run the project
